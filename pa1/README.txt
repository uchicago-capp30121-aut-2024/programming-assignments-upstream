Programming Assignment #1
-------------------------

These are the files for Programming Assignment #1.

- sir.py: Skeleton code. You will write your code in this file.

- test_sir.py: The automated tests. 
  Run the automated tests without vaccines: 

  $ py.test
  
  Run the automated tests with vaccines: 

  $ py.test --runvax

  See the "Testing Your Code" page on Canvas for
  more detailed instructions. 

- conftest.py: pytest configuration file

- pytest.ini: Configuration files that you can safely ignore.

- tests/: A directory containing files specifying the tests.

- sample_cities/: A directory with a few sample city files.

- README.txt: This file.
